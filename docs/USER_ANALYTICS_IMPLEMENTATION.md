## üîê SECURITY & PRIVACY

### Encryption Details

```
User ID (UUID)
  ‚Üí SHA-256 Hash
  ‚Üí 256-bit AES Key
  ‚Üí AES-256-GCM Encryption
  ‚Üí Server stores ciphertext only
  ‚Üí Client decrypts with same key
```

**Security Properties:**

- ‚úÖ **Confidentiality**: Server can't read URLs
- ‚úÖ **Integrity**: Tamper detection (GCM tag)
- ‚úÖ **Authenticity**: Each encrypted message authenticated
- ‚úÖ **Forward secrecy**: Unique nonce per encryption
- ‚úÖ **Zero-knowledge**: Server has no decryption key

### Privacy Features

- **No plaintext storage**: All URLs encrypted at rest
- **Hashed indexing**: Search without decryption
- **User control**: Export or delete all data
- **Data retention**: Auto-delete after 90 days (configurable)
- **No third-party tracking**: All analytics stay in your database
